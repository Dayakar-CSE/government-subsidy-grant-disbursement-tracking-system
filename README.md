# Government Subsidy & Grant Disbursement Tracking System

## Infosys Springboard Virtual Internship 7.0

### Project Status

Project Planning Phase

### Objective

Develop a centralized web application to automate the complete lifecycle of government subsidy and grant disbursement, including beneficiary registration, eligibility verification, approval workflow, fund disbursement, and reporting.

### Technology Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Spring Security
- REST APIs
- Maven
- Git
- GitHub
